# Python-poker-tool

An initial NEA project developed into a python-based poker simulator.

---

## Overview

This application allows for users to create a login, sign in using this login, and play poker rounds against a simple AI opponent. It allows for saved progression, and aims to teach new users how to play Texas Hold'em.

## How to Run

python main.py

## Setup
pip install -r requirements.txt
python mainprogram.py
